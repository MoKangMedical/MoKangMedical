## 🏗️ Core Projects

| Project | Description | Tech |
|---------|-------------|------|
| [**OpenClaw Medical Harness**](https://github.com/MoKangMedical/openclaw-medical-harness) | AI Agent orchestration framework — 5-step harness pipeline | Python, MCP, PyPI |

| [**MediChat-RD**](https://github.com/MoKangMedical/medichat-rd) | Multi-agent rare disease diagnosis | CrewAI, PubMed, OMIM |
| [**MediPharma**](https://github.com/MoKangMedical/medi-pharma) | AI drug discovery platform | ChEMBL, OpenTargets, ADMET |
| [**DrugMind**](https://github.com/MoKangMedical/drugmind) | Drug R&D digital twin collaboration | Digital Twin, RWD |
| [**云思园**](https://github.com/MoKangMedical/cloud-memorial) | AI memorial — digital legacy & grief therapy | Voice Cloning, LLM |
| [**数字智者**](https://github.com/MoKangMedical/digital-sage) | Chat with 100+ historical philosophers & scientists | 100+ AI Agents |
